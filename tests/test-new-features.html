🧪 HP-12C New Features Test

✅ Test Plan

These features were just implemented and need testing:

  • CHS (Change Sign) - Negate values
  • STO (Store) - Store to memory registers
  • RCL (Recall) - Recall from memory registers

Test Cases

1. CHS - Change Sign

Steps:

  1. Enter 5
  2. Press CHS
  3. Display should show -5
  4. Press CHS again
  5. Display should show 5

Expected: Sign toggles between positive and negative

2. CHS During Entry

Steps:

  1. Enter 1 2 3
  2. Press CHS
  3. Display should show -123

Expected: Sign changes during digit entry

3. STO - Store to Memory

Steps:

  1. Enter 42
  2. Press STO
  3. Press 5
  4. Console should log: "Stored 42 to R5"
  5. Display should still show 42 (non-destructive)

Expected: Value stored in register 5, X unchanged

4. RCL - Recall from Memory

Steps:

  1. With 42 in R5 from previous test
  2. Enter 0 (to see it's different)
  3. Press RCL
  4. Press 5
  5. Display should show 42
  6. Console should log: "Recalled 42 from R5"

Expected: Value recalled to X with stack lift

5. STO + RCL Combo

Steps:

  1. Enter 100
  2. Press STO 0
  3. Enter 200
  4. Press STO 1
  5. Press CLx (clear display to 0)
  6. Press RCL 0 → should show 100
  7. Press RCL 1 → should show 200
  8. Press + → should show 300

Expected: Memory system enables complex calculations

6. Stack Lift with RCL

Steps:

  1. Clear and enter 10 ENTER 20 ENTER 30
  2. Stack: T=10, Z=20, Y=30, X=30
  3. Store something: 99 STO 9
  4. Then: RCL 9
  5. Stack should lift: T=20, Z=30, Y=30, X=99

Expected: RCL causes proper stack lift
